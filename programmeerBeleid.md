In general: not all existing code adheres to this policy (this is an understatement). The idea is that **new** code should be compliant, and whenever you change old code leave it more compliant than you found it!

  * [Documentation](Documentation.md): Every module, class, and 'public' method should have a doc string.
  * UnitTests: Unit tests should be used where possible to prevent regression.
  * copyright: Please put the copyright notice and license above every file.
